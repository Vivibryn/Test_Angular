Mission du stage

Recupere des données depuis notre serveur rest et gerer le traitement & l'affichage
Dans l'idée la mission sera de moderniser cette page ci (armaor shape) avec la solution angular.

Information sur le test

J'ai préparer sur notre serveur angular une partie que tu pourra contacter pour effectuer le test.
Le but sera de recuperer un object json depuis notre server rest pour ensuite l'afficher et faire du traitement.
  • Communiquer avec un serveur rest
  • Recuperer & afficher le resultat de cette communication
  • Faire du traitement sur ce resultat (habilage, calcule)

Ex 1:

La premier fonction à appeler sera "GetAllGrade"
Elle te retournera un tableau en JSON Contenant les champs suivant :
{{'
{
"Name": string,
"ShortName": string,
"Code": string,
"Default": number
}
'}}
Je veux que tu sois capable d'appeler la fonction (au lancement de la page ou avec un evenement sur un btn).
Et pouvoir afficher les informations depuis sur un composant angular (celle ci par exemple).
Astuce:
Sur angular je te conseil de regarde le *ngFor="let item of items" sur une balise pour faire une boucle d'un
tableau
et en plus complex tu a le mat-table .

Ex 2:

Je voudrais que tu appel la fonction "GetDiamFromGrade" Pour recuperer le poids linéaire par diameter d'une nuance
d'acier en passant en parameter la nuance de l'acier
Ex (185.162.208.123:3002/datasnap/rest/TServerMethods/GetDiamFromGrade/T)
En plus de ceci il faudrais que l'appel de la fonction soit dissimuler dans le tableau des nuance et qu'on en fasse
l'appel en cliquand sur un bouton.
Restriction :
  • Ecrire le code js dans le composant angular (ici app.component.ts)
  • Utiliser Angular material pour l'affichage (pas de bootstrap)

Info / AIDE

Pour se connecter a notre serveur

  • IP: 185.162.208.123
  • PORT: 3002
  • CHEMIN D'ACCES: /datasnap/rest/TServerMethods/"LE NOM DE LA FONCTION"/"Argument1"/"Argument2"/"ArgumentX"

Exmaple requete rest

http://185.162.208.123:3002/datasnap/rest/TServerMethods/EchoString/toto
Cette exemple call le serveur reste la fonction "EchoString" de la class "TServerMethods" avec comme parametre "toto".
